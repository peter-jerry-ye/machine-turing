Mini-projet IF2A : La Machine de Turing
ASINSA 96 - VANNIER Corentin, WANTZ Gabriel, WONG Teck Wan, YE Zihang

Structure de notre archive :
/src : le répertoire contenant le code source du projet
/exemplesTA : le répertoire contenant des exemples de machine de Turing sous format texte
/images : le répertoire contenant les images utilisées dans l'IHM

Pour éxecuter le projet :
- Compiler les fichiers .java contenus dans le répertoire src (javac Fenetre.java)
- Executer le fichier Fenetre.class (java Fenetre)
